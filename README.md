## ‚ö° About The Project

**Varaha Labs Studio** is a cutting-edge digital agency portfolio designed to showcase high-end web development, motion graphics, and AI solutions. Built with a focus on **performance**, **aesthetics**, and **user experience**, this project pushes the boundaries of modern web design.

### üåü Key Features
-   **Immersive UI/UX**: Glassmorphism, neon aesthetics, and smooth GSAP animations.
-   **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop.
-   **Secure Authentication**: Integrated Google Sign-In via Firebase Auth.
-   **Real-time Database**: Cloud-synced forms using Firebase Firestore.
-   **Interactive Elements**: 3D backgrounds with Three.js and custom cursor effects.
-   **AI Integration**: "Nexus AI" concept for generating project blueprints.
